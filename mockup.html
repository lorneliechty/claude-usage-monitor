Architecture

Stack: Python 3 + rumps + py2app → native .app bundle

Data source: Session cookie auth → claude.ai internal API (same endpoints the settings page uses)

Polling: Every 60s (configurable), with immediate refresh on click

Notifications: Native macOS via osascript or pync

Auth: One-time session cookie paste from browser, stored in macOS Keychain

Size: ~5 MB bundled app
